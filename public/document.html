Overview

The GOM (Game of Mito) Testnet Stats Viewer is a comprehensive analytical tool that enables users to examine their testnet statistics and explore potential allocation scenarios based on their on-chain activity within the Mitosis testnet environment.

Core Purpose

This application assists participants in visualizing their on-chain interactions and understanding how various allocation parameters might influence potential reward distributions.

Note: This is an independent, community-developed analytical tool not affiliated with the official Mitosis team. All computations represent estimates intended for educational purposes.

Asset Reconciliation Methodology

The application calculates net asset values using this established formula:

net_assets = Î£(vanilla + aTest + mi - variableDebtTest)token

Where:

  • vanilla tokens: Standard token balances without protocol-specific prefixes
  • aTest tokens: Assets supplied to testnet protocols (prefixed with "aTest")
  • mi tokens: Mirrored assets (prefixed with "mi") representing EOL deposits
  • variableDebtTest tokens: Borrowed assets within testnet protocols

For Ethereum (ETH), the calculation process is:

This methodology is consistently applied across all supported assets.

Usage Guide

Basic Operation

  1. Input your wallet address (0x... format)
  2. Specify the testnet allocation percentage (default: 3.5%)
  3. Select "View Testnet Stats" to initiate analysis
  4. Review comprehensive results dashboard
  5. Optionally share or download results

Advanced Configuration

API Reference

The application integrates with the following Routescan API endpoints:

MITO Balance:
GET https://api.routescan.io/v2/network/testnet/evm/124832/etherscan/api?module=account&action=balance&address={walletAddress}&tag=latest
wMITO Balance:
GET https://api.routescan.io/v2/network/testnet/evm/124832/etherscan/api?module=account&action=tokenbalance&contractaddress=0x8B71fD76b23721d0D73CA14Ea6464AAD70149B67&address={walletAddress}&tag=latest
ERC-20 Holdings:
GET https://api.routescan.io/v2/network/testnet/evm/124832/address/{walletAddress}/erc20-holdings
Transaction Metrics:
GET https://api.routescan.io/v2/network/testnet/evm/all/address/${walletAddress}/transactions?includedChainIds=124832&count=true&limit=0
